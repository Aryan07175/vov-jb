# VOV JB

A replica or styled version of VOV.VN, incorporating designs from VnExpress.

**Live Demo:** [https://vov-vn-replica.vercel.app](https://vov-vn-replica.vercel.app)

## Files
- `index.html`: The main HTML file containing the structure of the page.
- `vov-vnexpress-style.css`: The CSS stylesheet containing the custom styles for the page.
